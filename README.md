# Hangman Game (C)

A console-based Hangman game implemented in C, focusing on control flow, string handling, and user interaction.

## Features
- Word guessing gameplay
- Random word selection
- Letter-by-letter guessing
- Limited attempts
- Win / lose detection

## How to Build
```bash
gcc src/hangman.c -o hangman

## Concepts Used
- Loops and conditionals
- Arrays and strings
- Functions
- Input validation
